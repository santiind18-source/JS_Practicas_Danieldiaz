JavaScript - Estructuras de Clase

9.1 Fundamentos de la Sintaxis de Clases

Exploración de la declaración básica de una clase en JavaScript y cómo se instancian objetos a partir de ella, sentando las bases de la programación orientada a objetos.

9.2 Herencia entre Clases

Demostración práctica de cómo una clase puede extenderse de otra, heredando sus propiedades y métodos. Se ilustra la relación "es un" fundamental en el diseño orientado a objetos.

9.3 Miembros Estáticos: Propiedades y Métodos

Análisis de las propiedades y métodos que pertenecen directamente a la clase constructora, en lugar de a sus instancias individuales. Son utilidades asociadas a la clase en sí, no a los objetos específicos.

9.4 Control de Acceso: Miembros Privados y Protegidos

Este segmento ilustra el uso de convenciones como el prefijo `_` para miembros "protegidos" y el `#` para miembros "privados" (ecmascript privado real), para gestionar el nivel de acceso a las propiedades y métodos dentro de una clase.

9.5 Extensión de Clases Nativas

Ejemplo práctico de cómo extender y personalizar clases preexistentes de JavaScript, como `Array`, `Map` o `Error`, adaptando su funcionalidad a necesidades específicas del proyecto.

9.6 Verificación de Instancia: el operador "instanceof"

Uso del operador `instanceof` para verificar de forma fiable si un objeto es una instancia directa o indirecta de una clase específica o de un constructor prototípico.

9.7 Mixins: Composición de Funcionalidades

Demostración de la aplicación de mixins como patrón para incorporar funcionalidades reutilizables a las clases, permitiendo una composición flexible y evitando la herencia múltiple.
